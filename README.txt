LOAD TRACKER PRO — OPERATIONS UPDATE

WHAT CHANGED
- The app is now framed around tracking shipment loads and driver updates.
- Metrics emphasize loads tracked, not taps.
- Each positive update counts as about 5 minutes of work.
- The app shows:
  - Loads tracked today
  - Estimated time spent updating loads
  - Current hourly load pace
- The hourly racecourse is placed at the top by default.
- The +5 and -5 buttons were removed.
- Major sections can be reordered.
  - On Mac/desktop: drag a section using its DRAG handle.
  - On phone: use the up and down arrow buttons.
- Your chosen section order is saved locally.
- Existing load history, stats, achievements, horses, race wins, and settings remain stored locally.

HOW TO UPDATE GITHUB
1. Unzip this folder.
2. Open your existing Pulse-Counter-Pro repository.
3. Click Add file → Upload files.
4. Drag all files from this folder into GitHub.
5. Replace the existing files.
6. Click Commit changes.
7. Wait about 1–2 minutes.
8. Reload your GitHub Pages app.

IF THE OLD VERSION APPEARS
- Refresh twice.
- Or fully close and reopen the installed app.
- The service-worker cache version was updated.

NOTE
The GitHub repository may still be named Pulse-Counter-Pro. That is fine.
The app itself will display as Load Tracker Pro.
